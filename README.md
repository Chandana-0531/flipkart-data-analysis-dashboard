## Features
- Product category analysis  
- Price distribution  
- Rating distribution  
- Discount percentage analysis  
- Brand comparison  
- Search products  
- Filter by price, rating, and category  
- Download filtered results as CSV  

## Technologies Used
Python, Pandas, Seaborn, Matplotlib, Streamlit
